Memo_er: flexible self-service data analysis

Memo_er is temporarily down while being revised and released as open source!

Here's a short blurb on it:

Memo_er is a self-service data analysis tool, an automated memo-writer. Like a dashboarding system, it takes inputs from a user, runs an analysis written by a data scientist, and formats the results. Unlike dashboarding systems, it allows the data scientist to write the analysis in Python, allowing more complex analyses to be automated, and presents results as memos, the standard output of data analysis, including explanations of the premises of the analysis and how to interpret the conclusions.

Self-service data analysis tools allow Data Scientists to focus on their comparative advantage: translating a set of inputs or Premises and Data into a Conclusion, while allowing Product and other business units to decide exactly what the Premises of an analysis should be and how the Conclusions of an analysis should impact business decisions. This saves time, produces higher quality, reproducible work output, and enables Data Scientists to scale their impact.
